
Premissas para executar o script com efetividade:

1-Utilize um usuário com permissões de Admin e Log on as a service, defina ele dentro do script AtualizaAgentHorizon.bat

2-Coloque os arquivos Omnissa-Horizon-Agent-8.18.exe e Install.bat em alguma pasta da rede que o destino tenha permissão de acesso.

3-Coloque os hostnames ou IPs dos VDI, no arquivo ListaAtualizaVDI.txt, um abaixo do outro, ex:

VDI-001
VDI-002
VDI-003

4-Execute o script AtualizaAgentHorizon.bat via CMD ou PowerShell como Administrador
